MCLite Web Flasher

Flash MCLite firmware to your T-Deck Plus directly from the browser

Your browser does not support Web Serial. Please use Chrome, Edge, or another Chromium-based browser.
Version
Web Serial not supported in this browser. Web Serial access denied. Ensure the page is served over HTTPS.
Instructions
1. Switch off the T-Deck Plus
2. Press and hold the trackball button
3. Switch the device on while holding the trackball
4. Release the trackball — the device is now in DFU mode
5. Connect via USB and select a firmware version above
6. Click "Install MCLite" and select the serial port
7. Wait for the flash to complete (~30 seconds)

First-time flash? You will be prompted to erase the device. This is normal.
Note: Your config.json and SD card contents are not affected by flashing.
